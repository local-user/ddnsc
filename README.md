# ddnsc
'dynamic dns cloudflare'

## overview
DDNSC updates a CloudFlare 'A' record with a retrieved IP.

## feature(s)
- display
- update

## (optional) config
- Create    '~/.config/ddnsc'
- Copy      'sample/CloudFlare.cfg' -> '~/.config/ddnsc'
- Edit      '~/.config/ddnsc/CloudFlare.cfg'
- Run

## note(s)
Created because ISP does not allow static IPs.
